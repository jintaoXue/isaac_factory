# 运行方式

## 需要流媒体传输

1. 服务器终端1：

```
conda activate env_isaaclab
python train.py \
  --task HRTPaHC-v1 \
  --algo rule_based \
  --num_envs 1 \
  --active_livestream \
  --livestream_public_ip 10.68.14.234 \
  --livestream_port 49100 \
  --device cuda:0 \
  --headless
```

2. livestream client捕捉实时仿真画面

或  

服务器终端2：

```
cd ~/work/isaac_factory/source/isaaclab_tasks/isaaclab_tasks/direct/hc_factory/output
python3 -m http.server 8888 --bind 0.0.0.0
```

3. MES前端：
```
pnpm dev
```

## 只做训练

服务器终端

```
python train.py --task HRTPaHC-v1 --algo rule_based --num_envs 1 --device cuda:0 --headless 
需要采集视频(mp4)的话可加：--video --video_length 10000 --video_interval 10000
```

## 注入瓶颈扰动（四选一维度）

```
python train.py --task HRTPaHC-v1 --algo rule_based --num_envs 1 --device cuda:0 --headless \
  --disturbance_dim machine \
  --disturbance_intensity 1.0
```

`--disturbance_dim`：`none` | `material` | `human` | `logistics` | `machine`  
详见 [`03.实验设计_瓶颈维度与扰动.md`](./03.实验设计_瓶颈维度与扰动.md)。
